## Selected Projects

<details>
<summary><strong>Small Language Model — Edge-Native Offline Diagnostics</strong></summary>
<br>

Offline industrial SLM diagnostic assistant deployed on **Raspberry Pi 4** — no cloud, no internet, no compromise.

- Integrated **Phi-3 Mini** + `llama.cpp` + **ChromaDB** + RAG pipeline
- Built for air-gapped industrial facilities with strict connectivity constraints
- RAG MRR: **0.83** | Time-to-First-Token: **2.4s** on constrained hardware
- Stack: `Python` · `llama.cpp` · `ChromaDB` · `LangChain` · `Raspberry Pi 4`
</details>

<details>
<summary><strong>NyayaGyan — AI-Powered Legal Assistant</strong></summary>
<br>

Role-based legal intelligence platform for document drafting and case-law retrieval.

- Hybrid search combining **BM25 + vector embeddings** for accurate retrieval
- Graph-aware legal reasoning via **Neo4j** knowledge graph
- Multi-user support with **PostgreSQL pgvector** for semantic similarity
- Stack: `LangChain` · `LangGraph` · `Neo4j` · `PostgreSQL pgvector` · `FastAPI`
</details>

<details>
<summary><strong>GMNE — Generative Multi-Physics Neural Emulator</strong></summary>
<br>

Physics-Informed Neural Network (PINN) + DDPM to accelerate hydrogen electrolyzer simulations — replacing slow CFD/FEM solvers.

- Achieved **R² > 0.94** vs. ground-truth CFD simulations
- Inference latency: **< 100ms** (vs. hours for traditional solvers)
- Integrated uncertainty estimation for trustworthy predictions
- Stack: `PyTorch` · `PINNs` · `DDPM` · `NumPy` · `SciPy`
</details>

<details>
<summary><strong>Autonomous KYB Agent — AI Due Diligence System</strong></summary>
<br>

Agentic AI system for automating corporate Know-Your-Business (KYB) due diligence in corporate finance.

- Multi-agent orchestration with automated document extraction and analysis
- Stack: `LangGraph` · `Python` · `FastAPI` · `LLMs`
</details>
